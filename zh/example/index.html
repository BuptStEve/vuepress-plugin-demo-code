# 常规操作

直接把 demo 代码用 ::: demo 和 ::: 包裹。即可生成可运行的 demo 和代码。

# 使用示例

::: demo
<template>
    <button class="animated shake infinite" @click="onClick">Click me!</button>
</template>

<script>
export default {
    methods: {
        onClick: () => { window.alert(1) },
    },
}
</script>

<style>
button {
    color: blue;
}
</style>
:::

    this is <span style="color: red;">common</span> html
</p>

    color: green;

</style>
