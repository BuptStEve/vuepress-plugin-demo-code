# Example

# Common Usage

Just wrapper your demo code between ::: demo and :::. Then you will get runnable demo and code.

# Usage Example

::: demo
<template>
    <button class="animated shake infinite" @click="onClick">Click me!</button>
</template>

<script>
export default {
    methods: {
        onClick: () => { window.alert(1) },
    },
}
</script>

<style>
button {
    color: blue;
}
</style>
:::

# Result
