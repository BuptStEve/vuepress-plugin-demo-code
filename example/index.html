# Eaxmple

# 常规操作

直接把 demo 代码用 ::: demo 和 ::: 包裹。即可生成可运行的 demo 和代码。

# 使用示例

::: demo
<template>
    <button @click="onClick">Click me!</button>
</template>

<script>
export default {
    methods: {
        onClick: () => { window.alert(1) },
    },
}
</script>
:::
