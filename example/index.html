# Common Usage

Just wrapper your demo code between ::: demo and :::.Then you can generate runnable demo and code.

# Usage Example

::: demo
<template>
    <button @click="onClick">Click me!</button>
</template>

<script>
export default {
    methods: {
        onClick: () => { window.alert(1) },
    },
}
</script>
:::
